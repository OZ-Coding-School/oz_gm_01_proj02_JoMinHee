# 던전 로그 (Dungeon Log)

2D 턴제 팀배틀 로그라이크 게임

## 프로젝트 개요

- **장르:** 2D 턴제 팀배틀 로그라이크
- **플랫폼:** PC (Windows), Android
- **개발 기간:** 3주
- **개발 목적:** 포트폴리오용 게임 제작

## 핵심 게임 플레이

- **랜덤 스킬 시스템:** 턴마다 캐릭터가 보유한 스킬 중 1개가 랜덤 추첨
- **리롤 시스템:** 스킬 1회 리롤 가능
- **팀 공유 AP:** 생존한 팀원 수만큼 AP 부여, 팀 전체가 공유
- **독립적 전투 계산:** 캐릭터/스킬/아이템/유물이 각각 객체가 되어 한 번에 계산
- **완전 로그라이크:** 런 성공/실패 후 얻은 재화로 가챠해서 새로운 캐릭터/아이템 해금

## 던전 구조

- **총 4층, 20전투** (일반 16 + 보스 4)
- 각 층: 전투 4회 + 구역 보스 1회
- 선형 진행 (전투/이벤트/상점/휴식 랜덤 배치)

## 기술 스택

- **엔진:** Unity 6000.3.2f1 LTS
- **언어:** C#
- **데이터 관리:** ScriptableObject + CSV
- **UI:** Unity UI (TextMeshPro)

## 프로젝트 구조

```
Assets/
├── 01.Scripts/          # C# 스크립트
├── 02.Prefabs/          # 프리팹
├── 03.Scenes/           # 씬
├── 04.Art/              # 아트 리소스
├── 05.ScriptableObjects/# SO 데이터
├── 06.Audio/            # 오디오
├── 07.Materials/        # 머티리얼
├── 08.Data/             # CSV 데이터
└── 09.Documents/        # 기획서
```

## 개발 로드맵

- **1주차:** 프로젝트 기반 구축, SO 데이터 시스템, 전투 계산 메니저
- **2주차:** 전투 시스템 완성, 캐릭터 시스템, 스킬 시스템
- **3주차:** 던전/이벤트 시스템, 뽑기 시스템, UI/UX 완성
